1. ::after создаёт псевдоэлемент после содержимого элемента. Этот псевдоэлемент работает только совместно со стилевым свойством content, которое определяет содержимое для вставки. Не у всех элементов есть собственное содержимое, поэтому ::after не даёт результата для , и ряда других элементов.; 2. pointer-events: none; - дает возможность пропустить через него собитие или клики (используеться для модалокб выпадаещего меню и т.д.). Когда бекдроп скрыт, нужно, чтобы он через себя пропускал все клики и другие события - решение pointer-events 3.user-select: none; - свойство для запрета выделения контента
